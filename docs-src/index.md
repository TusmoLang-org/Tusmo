# Tusmo Documentation

Welcome to Tusmo, a Somali programming language!

## Quick Start

```bash
# Install
curl -fsSL https://raw.githubusercontent.com/Tusmolang/Tusmo/main/install.sh | bash

# Run
tusmo hello.tus
```
